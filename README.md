# East Meets Feed — @Yujia

**NYU Social Media & Brand · June 2026**

A hybrid presentation website for the *East Meets Feed* cross-platform brand activation campaign for lifestyle creator @Yujia.

## Live Site

[View Presentation →](https://eric5193-web.github.io/yujia_Finalbp_deck/)

## About

This site documents a 7-part social media activation strategy spanning Instagram, TikTok/Douyin, and Xiaohongshu. It features two viewing modes:

- **Slides** — Fullscreen deck with keyboard navigation (← →)
- **Story** — Scrollable long-form version with sidenav
